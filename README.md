# Bobby Devarapu - Personal Portfolio

A modern, responsive portfolio website showcasing my projects, experience, and skills as a full-stack developer.

🔗 **[Live Demo](https://bobbyportfolio.vercel.app)**

## 🎨 Features

- ✨ Modern, responsive design with smooth animations
- 🎯 Sections: Hero, About, Experience, Education, Projects, Tech Stack, Contact
- 📱 Mobile-first approach with Tailwind CSS
- 🌙 Glass-morphism UI components
- ⚡ Fast performance with Vite
- 🎪 Interactive elements with React

## 💻 Tech Stack

- **Frontend Framework:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** shadcn-ui
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/bobbydevarapu/bobby-portfolio.git

# Navigate to project directory
cd bobby-portfolio

# Install dependencies
npm install
```

### Development

```bash
# Start the development server
npm run dev
```

The app will be available at `http://localhost:8082` (or the next available port)

### Build for Production

```bash
npm run build
```

## 📝 Customization

### Edit Locally
1. Clone the repository
2. Make changes to files in `src/components/`
3. Test with `npm run dev`
4. Commit and push to GitHub

### Edit on GitHub
1. Open the file in the repository
2. Click the pencil icon
3. Make your changes
4. Commit directly to main

### Use GitHub Codespaces
1. Click **Code** → **Codespaces** → **New codespace**
2. Edit in the browser IDE
3. Commit and push changes

## 🌐 Deployment to Vercel

```bash
# Install Vercel CLI
npm install -g vercel

# Login to Vercel
vercel login

# Deploy
vercel
```

Follow the prompts to:
- Link to your GitHub repository
- Select the project folder
- Install dependencies
- Deploy

Your live URL will be generated automatically (e.g., `https://bobbyportfolio.vercel.app`)
